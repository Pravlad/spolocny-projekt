JavaScript

Úvod a základná charakteristika

  • Bol vyvinutý firmou NetscapeCorporation (v spolupráci so Sun Microsystem)
  • Tvorcom skriptovacieho jazyka je BrendanEich
  • Vznikol v 2. polovici 90. - tych rokov minulého storočia
  • Je to jazyk interpretovaný, to znamená pokiaľ v ňom urobíme chybu, zistíme ju len v prípade, že používame prehliadač umožňujúci hlásiť chyby v skriptoch na webových stránkach
  • Funguje na strane klienta
  • Je objektovo orientovaný
  • Je nezávislý na platforme
  • Základná syntax pripomína C/C++ či mnohé iné podobné jazyky ako Java, C#, PHP, PERL
  • JavaScript je však na rozdiel od väčšiny menovaných programovacích jazykov beztypovým jazykom

    • Vloženie skriptu do HTML dokumentu

    JavaScriptový program môžeme do www stránky vložiť pomocou HTML taguscript. Napr.:

    <script type=“text/javascript“>
    /* ... programový kód ... */
    </script>

    Možnosti včlenenia:

  • do hlavičky
  • do tela HTML dokumentu
  • do samostatného súboru a tento potom prilinkovať k www stránke napr. <script type=“text/javascript“ src=“main.js“></script>
  • priamo do tagu pre ošetrovanie udalostí v niektorých objektoch

    • Prehliadač skript vykoná v momente, keď ho načíta.

    DOM – objektový model dokumentu

    Na čítanie dát z dokumentu, ako aj na jeho ľubovoľnú úpravu, sa používa tzv. DOM (z angl. DocumentObject Model, t.j. objektový model dokumentu). DOM nám ponúka kolekciu objektov, ktoré reprezentujú prehliadačom aktuálne načítaný dokument. S týmito objektmi môžeme pracovať podľa ľubovôle, a teda ich aj meniť. Kolekciu DOM objektov zastrešuje objekt document.
