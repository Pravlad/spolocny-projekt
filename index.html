Základné technológie na tvorbu www stránok

Medzi najpoužívanejšie technológie pre tvorbu www stránok dnes patria: HTML (XHTML), CSS, PHP a JavaScript.

Stránky vytvorené len pomocou HTML jazyka sa nazývajú statické. Takáto stránka sa každému návštevníkovi a v každom čase zobrazí rovnako.

Stránky, ktoré sa môžu dynamicky generovať napr. v závislosti od času alebo činnosti návštevníka nazývame dynamické.

HTML (XHTML)

HTML je značkovací jazyk. Značkuje (označuje) časti dokumentu, stránky.

To znamená, že pomocou HTML určujeme, akývýznam bude mať tá ktorá časť HTML stránky.

Pomocou HTML napr. povieme, že tento kus textu bude nadpis, iný kus textu odsek a inýzas tabuľka.

CSS

CSS je formátovací jazyk. To znamená, že určuje, ako bude ktorá časť stránky vyzerať. Pomocí CSS štýlu teda môžeme určiť vzhľadstránky.

CSS tedahovorí, že nadpis (o ktorom HTML určilo, že je nadpis) bude napr. modrý, že odseksa bude zarovnávaťna stred a že tabuľka bude na zelenom pozadí.

Naopak užnehovorí, ktorá časť stránky bude tým nadpisom, odsekom či tabuľkou.

PHP

PHP je skriptovací jazyk na strane servera.

Robia sa pomocou neho rôzne webové aplikácie ako je kniha hostí, počítadlo návštev, anketa, diskusné fórum alebo chat.

JavaScript

JavaScript je skriptovací jazyk na strane klienta.

To znamená, že to, čo robí JavaScript sa vždy odohráva na našom počítači (resp. na počítači užívateľanašich stránok) a neodosiela žiadne dáta na server.

. Pomocou JavaScriptu teda nevytvoríte nič, čo by muselo prebehnúť Internetom.

JavaScripty dokonca fungujú, aj keď niesme k Internetu práve pripojení.
